Notable Projects

Intelligent Tic-Tac-Toe

Designed and trained a machine learning algorithm capable of advanced player-vs-computer tic-tac-toe gameplay. Implementation inspired by weeks of online research and can be easily adapted to other player-vs-computer games. Algorithm can achieve a win rate of up to 95% when playing against randomly generated machine agents. Developed for grade 12 computer science class.

Objectively Best Calendar

Part of a team of 5 that developed a useful calendar computer application. Application allows users to:

  • Log in/create users.
  • Display an advanced graphical user interface calendar inspired by Google Calendar.
  • Create/edit/delete alerts/events/memos.
  • Send events to other users.
  • View a 5 day weather forecast.
  • Notify the user of an event.

I learned important teamwork skills that will be appreciated for years to come. Developed for CSC207H1: Software Design.

Graphing Calculator

Developed a graphing calculator application along with 2 peers capable of parsing and displaying functions. Graphical user interface developed using javax.swing and parser was developed from scratch. Project developed for AP calculus BC class in grade 12.

String Genetic Algorithm

Researched combinatorial potential of a genetic algorithm by developing an algorithm that guessed a string. Project was developed as component of Research-a-thon 2020 entry in a group of 3. Goal of the research was to develop strategies to minimize runtime of a genetic algorithm with a select number of reinforcers. Further research will address larger sample spaces. Source code can be found here.

Grant Theft Autopilot

Developing a model in tensorflow that will be able to drive vehicles in Grand Theft Auto 5. Currently in development. Updates will be posted as progress is made.
